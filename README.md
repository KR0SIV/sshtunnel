sshtunnel
=========

ssh socks tunnel
this tunnel can auto reconnect

INSTALL&RUN
-------
1. pip install paramiko
1. pip install sshsocks
1. sshsocks -C to init ~/.config/sshsocks.conf
1. edit $HOME/.config/sshsocks.conf write your account info
1. run sshsocks.py
